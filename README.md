# PR2_Logic_Box
# 🌟 Pattern Generator & Number Analyzer

A beginner-friendly Python project that combines **pattern generation** and **number analysis** in a simple, interactive menu-driven application.

---

## ✨ Features

### 🎨 Pattern Generator

* Generate a beautiful right-angled star (`*`) pattern.
* User can choose the number of rows.

### 🔢 Number Analyzer

* Analyze numbers within a selected range.
* Identify each number as **Even** or **Odd**.
* Calculate the **sum** of all numbers in the range.

### 📋 Interactive Menu

* Easy-to-use menu-driven interface.
* Perform multiple operations without restarting the program.
* Exit anytime using the Exit option.

---

## 🚀 What You'll Learn

* ✅ Variables and User Input
* ✅ `if`, `elif`, and `else` Statements
* ✅ `while` and `for` Loops
* ✅ Nested Loops
* ✅ Pattern Printing
* ✅ Even/Odd Logic
* ✅ Sum Calculation

---

## 📌 Application Flow

```text
Welcome to the Pattern Generator and Number Analyzer!

1️⃣ Generate a Pattern
2️⃣ Analyze a Range of Numbers
3️⃣ Exit
```

The application keeps running until the user selects **Exit**.
